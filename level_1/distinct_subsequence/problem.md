# Problem

### Statement
Given a string of length N, find the count of distinct subsequence of the given string.

Input format:

### Input format:
- N
- string

### Test Cases
1.
    #### Input:
    - 4
    - look
    #### Output:
    - 12
    #### Explaination:
    - "", "l", "o", "k", "lo", "lk", "oo", "ok","loo", "lok", "ook", "look" - total count 12.
